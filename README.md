# 👥 Cliente Service

Microsserviço responsável pelo gerenciamento de dados cadastrais dos clientes.

## 🛠️ Stack
- Spring Data MongoDB
- MongoDB (NoSQL)
- Spring Web

## ⚙️ Funcionalidades
- Cadastro completo de clientes com endereço.
- Consulta de clientes para suporte ao processo de venda.
